Detailed Description

Основной класс реализующий UDP сокет с возможностью приема данных из multicast шруппы.

Класс наследник QDialog, предоставляющий основную форму, содедержащий сокет для отправки/получения данных и содержащий элементы логики по асинхронному урпавлению воротами.
